SHCustomListPicker

A simple list picker shown as a popover view. It can display a list of strings sorted alphabetically with buttons: Cancel-to remove the view without selecting, Done-to return the selected item, Custom button- whose title and action can be customized. Also the list of strings can be indexed into alphabet ranges to quickly jump to a list of strings starting with the alphabet range.
